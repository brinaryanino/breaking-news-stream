# 📰 BNS — Breaking News Stream

BNS (Breaking News Stream) is a modern, minimalist news platform inspired by Medium, built to deliver a clean and distraction-free reading experience for Indonesian news.

This project is part of my portfolio, focusing on building real-world web applications using modern technologies and API integration.

---

## ✨ Features

- 📰 Latest Indonesian news (API integration)
- 🔍 Search articles by title
- ⭐ Bookmark system (localStorage)
- 🌙 Dark mode support
- ⚡ Fast and responsive UI
- 🧠 Clean reading experience (Medium-inspired)

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **API:** Berita Indo API
- **State:** React Hooks
